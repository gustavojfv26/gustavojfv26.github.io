Capítulo 13 – O Esconderijo

Você se esconde atrás de uma árvore caída.


A criatura passa lentamente pela clareira.


É um lobo gigantesco feito de sombras.


"Poucos respeitam a floresta. Se provar coragem, eu o ajudarei."


Quando ele vai embora, você percebe dois caminhos:


  • Um caminho que leva para um lago iluminado
  • Um caminho que leva para uma torre antiga

Escolha

  • Se seguir para o lago→vá para Capítulo 14
  • Se seguir para a torre→vá para Capítulo 18
